# 📚 AI小说生成器 (AI Novel Tool)

> **不止于生成，更是您的创作合伙人。**  
> 这是一个基于 Python 桌面端的 AI 辅助小说创作工具，专为网文作者、爱好者打造。它不仅能帮你瞬间生成万字大纲和正文，更提供独特的“**生成即起点，微调见真章**”的人机协作模式。
>
> 📦 **便捷下载**：[`ai-novel.zip`](release/ai-novel.zip) (Windows 打包版)

---

## ✨ 核心亮点：精细化“人机共创”流

本工具深度挖掘了 AI 在创作中的协同潜力，拒绝“黑盒式”一键生成。

### 1. 🪄 手术刀级微调 (AI Polishing)
- **全局指令**：输入“增加心理描写”、“让对话更幽默”，AI 即可全篇重塑。
- **内联式精准修改**：**黑科技功能！** 您可以在正文中直接插入 `【这里改为更激烈的打斗描写】` 或 `[[建议：主角此时应该感到犹豫]]` 等标记，点击“AI修改正文”，AI 将精准识别您的意图，仅针对标记处进行智能化重写并完美融合。

### 2. 🧩 深度语料库 (World Building)
- **动态设定加载**：AI 实时读取您定义的角色卡和世界观文档。
- **逻辑一致性**：无论是境界划分还是伏笔回收，AI 都能根据您的设定库保持叙事逻辑严丝合缝，告别“人设崩塌”。

### 3. 🧠 记忆生长系统 (Living Memory)
- **人物档案自动增量同步**：**突破性功能！** AI 会从每一章的定稿中自动提取角色的成长、关系变动及关键经历，并智能同步至人物设定卡中，实现“边写边记”，无需手动维护设定列表。
- **全局脉络动态整合**：系统会自动维护一份“全书梗概”和“近三章回顾”，确保创作到百万字时，AI 依然能精准掌握剧情走向。

---

## 🚀 主要功能一览

- **🤖 多模型支持**：DeepSeek (首选)、OpenAI、Gemini 一键切换。
- **📝 沉浸式编辑器**：集成了字数实时统计、章节管理、自动保存功能。
- **📖 智能定稿流程**：一键生成本章精炼摘要、更新全局摘要、并自动同步角色状态，构建完整的创作记忆链。
- **📄 极简导出**：支持导出为纯净的 TXT 文件，方便发布至各大小说平台。

---

## 🍱 创作流推荐

1.  **立项**：在“小说设置”中确定书名、题材与风格。
2.  **设定**：在“小说设定”中添加角色和背景。
3.  **策划**：在“章节大纲”中生成构思并**微调**。
4.  **初稿**：切换到“正文生成”，一键产出。
5.  **蜕变**：通过“修改要求”进行整体润色，或直接在文中插入 `【标记】` 进行局部微调。
6.  **定稿**：点击“生成定稿摘要”，自动完成记忆同步并定编章节。
